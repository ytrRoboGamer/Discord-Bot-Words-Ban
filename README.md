# Discord-Bot-Words-Ban
A Discord bot that will delete messages that contains any of the words you banned

--------------------

# How to use :
1. Download and instal Node.js

2. Download the files "index.js" and "bannedWords.json" and put them in the same file where you installed Node.js

3. Go to https://discord.com/developers/applications and create a bot

4. After creating a bot copy the token of the put

5. Open "index.js" and change PUT_YOUR_TOKEN_HERE to the token you copied

6. Put the words you want to ban in "bannedWords.json" (the words should be inside [ ] and between " " and seperated with commas and lower cases)

7. And finally you can run the code
If you are using VS Code :
- Select the file index.js and click Terminal -> New Terminal
- Then write node index.js

If you are not using VS Code :
- find the folder you put the file in
- on an empty place in the folder right click -> Open In Terminal
- write node index.js
